Tips untuk pempermudah penggunaan dan contoh pemanfaatan fitur yang biasa dilakukan.

## PDF

Anda dapat [mengupload file media pembelajaran](/docs/support/features#--upload-media) salah satunya adalah berbentuk PDF hanya dengan *drag and drop* ke ruangan tersebut. 

Jika anda berpindah halaman pada pdf anda pengguna lain juga akan melihat halaman yang sama.

Audien bisa mendownload file PDF tersebut dan screenshot halaman pdf yang sedang tampil.

Secara realtime anda dapat juga menggunakan [fitur pen](/docs/support/features#--drawing-pen) untuk melingkari sesuatu di PDF anda.

## Share Link

Untuk membagikan link anda dapat memanfaatkan [fitur chat](/docs/support/features#5-chat) atau bisa dengan *copy paste* langsung kedalam ruangan
